## About This WEBSITE

This is the template to create your own website portfolio.

## Installation Local on Windows

- config XAMPP (httpd-vhosts.conf)
- config windows hosts file
- clone the repo
- cd to repo
- remove .git and init new git repo
- composer install
- cp .env.example .env
- php artisan key:generate
- set up database
